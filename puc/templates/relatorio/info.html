Ajuda - Visão Relatório


Para que serve?


A visão relatório permite a visualização do comportamento
de um determinado evento no tempo. Para isso, basta você definir
um produto, um alarme, um monitor, a data inicial e data a final no
formulário de pesquisa da aplicação e clicar no botão .

Como dar zoom no gráfico?


O usuário pode usar a barra de rolagem (figura 1) localizada no rodapé do
gráfico para poder expandir ou reduzir a quantidade de informação visualizada.


Figura 1 - Barra de rolagem para ajustar o gráfico

Quais os formatos dos relatórios?


O formato do relatório gerado pode ser de 3 tipos diferentes, dependendo
do que está configurado na aplicação:
  1. Relatório Tabular
  2. Relatório com Gráfico de Linhas
  3. Relatório com Gráfico de Barras

Relátorio Tabular


É a forma mais simples de apresentação dos dados através de tabelas
sem qualquer tipo de interação.

Relatório com Gráfico de Linhas


Neste tipo de relatório os dados são apresentados em um gráfico de linha
com um certo grau de interação com o usuário. Pode-se posicionar o ponteiro
do mouse sobre um ponto no gráfico para visualizar os valores no eixo X e Y.
Além disso, o usuário pode usar a barra de rolagem (figura 1) localizada no rodapé do
gráfico para poder expandir ou reduzir a quantidade de informação visualizada.


Figura 1 - Barra de rolagem para ajustar o gráfico
